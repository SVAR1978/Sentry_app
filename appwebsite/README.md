# App Website

This folder is a simple static website for downloading an Android APK.

## Files

- `index.html` - landing page with the download button
- `styles.css` - page styling
- `app-release.apk` - replace this placeholder with your real APK

## How to use

1. Build your Android APK.
2. Rename it to `app-release.apk` or update the link in `index.html`.
3. Replace the placeholder file in this folder.
4. Upload the whole `appwebsite/` folder to a static host.

When a user taps **Download APK**, the browser downloads the linked file.
